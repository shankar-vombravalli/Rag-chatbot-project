📌 Overview

This project is a Retrieval-Augmented Generation (RAG) chatbot that answers questions based on documents using LLM.

⚙️ Technologies Used
Python
Sentence Transformers
FAISS / ChromaDB
Ollama (LLM)
Streamlit (UI)

📁 Project Structure
app.py → Main UI
rag_engine.py → Core logic
vector_store.py → Embeddings & storage
preprocessing.py → Data cleaning
🚀 How to Run
pip install -r requirements.txt
python app.py
